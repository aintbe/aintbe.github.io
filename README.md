# Hugo로 만든 블로그

## 글 작성하는 법

Hugo로 포스트를 작성하려면 말머리가 필요해요.

```markdown
+++
author = "aintbe"       # optional
title = "Guide"
date = "2019-03-05"     # optional
description = "Guide"   # optional
tag = ["tagname"]       # optional -> /tag/tagname
category = "categoryname"   # optional -> /category/categoryname
+++
```
